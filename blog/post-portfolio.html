Lancer un portfolio scientifique sur GitHub Pages

Article technique — 4 min de lecture

Un site personnel doit être rapide, simple et facile à maintenir. GitHub Pages est une bonne option lorsque l’on veut un site statique propre avec un domaine personnalisé.

Configuration

  • HTML, CSS statiques et JavaScript minimal.
  • Pages à la racine pour un déploiement direct sur GitHub Pages.
  • Fichier CNAME pour iliasbenkamoun.me.

Étape suivante

Le site pourra plus tard évoluer vers Astro ou Next.js si vous voulez une gestion plus dynamique du contenu scientifique et des publications.
